# St Cecilia Choir Financial Analysis (2025)

This project analyzes the income and expenses of St Cecilia Choir using R.

## Tools Used
- R
- dplyr
- ggplot2
- SQL concepts (GROUP BY, running totals)

## Tasks
- Data cleaning
- Expense grouping
- Income summaries
- Running balances
- Visualizations
- Financial dashboard

## Results
Income: ₦1,431,950
Expenses: ₦982,800
Surplus: ₦509,150

- choir_report.Rmd → analysis code
- Choir_Financial_Report.pdf → final report
